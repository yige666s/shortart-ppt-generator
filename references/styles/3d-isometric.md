# 3d-isometric

Three-dimensional isometric illustration style with depth and spatial design

## Design Aesthetic

Isometric 3D illustrations with consistent perspective, creating depth while maintaining clarity. Clean geometric forms with subtle shadows and highlights.

## Background

- Color: Soft Blue Gray (#E8EEF2)
- Texture: Subtle gradient, clean

## Typography

### Primary Font (Headlines)

Bold geometric sans-serif like Rubik Bold or Exo 2 Bold. Modern, tech-forward.

### Secondary Font (Body)

Clean sans-serif like Roboto or Noto Sans. Clear on 3D backgrounds.

## Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Background | Soft Blue Gray | #E8EEF2 | Primary background |
| Primary | Tech Blue | #4A90E2 | Main objects |
| Secondary | Mint | #50E3C2 | Supporting elements |
| Accent | Coral | #FF6B6B | Highlights |
| Shadow | Dark Blue | #2C3E50 | Depth, shadows |

## Visual Elements

- Isometric grid (30° angles)
- 3D geometric shapes
- Consistent light source (top-left)
- Soft shadows for depth
- Layered compositions
- Technical objects and interfaces

## Style Rules

### Do

- Maintain consistent isometric perspective
- Use uniform light direction
- Create depth through layering
- Keep shapes geometric and clean
- Use shadows subtly

### Don't

- Mix perspectives
- Use realistic textures
- Create overly complex scenes
- Add photographic elements
- Add slide numbers, footers, or logos

## Best For

Tech products, SaaS platforms, architecture, process flows, system diagrams, infrastructure visualization
