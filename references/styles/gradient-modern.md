# gradient-modern

Contemporary gradient-rich design with smooth color transitions and modern aesthetics

## Design Aesthetic

Modern, fluid design featuring smooth gradient transitions, glassmorphism effects, and contemporary color palettes. Creates depth and visual interest through layered gradients.

## Background

- Color: Gradient backgrounds (multi-color)
- Texture: Smooth gradients, glass-like overlays

## Typography

### Primary Font (Headlines)

Modern geometric sans-serif like Outfit or Space Grotesk. Clean, contemporary letterforms.

### Secondary Font (Body)

Neutral sans-serif like Inter or DM Sans. Excellent readability on gradient backgrounds.

## Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Gradient 1 | Purple to Pink | #8B5CF6 to #EC4899 | Primary gradients |
| Gradient 2 | Blue to Cyan | #3B82F6 to #06B6D4 | Cool tones |
| Gradient 3 | Orange to Red | #F97316 to #EF4444 | Warm accents |
| Text Light | White | #FFFFFF | On dark gradients |
| Text Dark | Deep Navy | #1E293B | On light backgrounds |

## Visual Elements

- Smooth multi-color gradients
- Glassmorphism cards with blur effects
- Soft shadows and depth
- Rounded corners on all elements
- Floating elements with subtle shadows
- Gradient text effects

## Style Rules

### Do

- Use smooth color transitions
- Create depth with layering
- Maintain high contrast for text
- Use rounded, soft shapes
- Apply subtle animations mentally

### Don't

- Use harsh color stops
- Create flat, single-color designs
- Overcomplicate with too many gradients
- Use sharp corners
- Add slide numbers, footers, or logos

## Best For

Tech startups, creative agencies, modern brands, app launches, design portfolios, innovation showcases
