# illustrated

Flat illustration style with clean vector graphics and friendly visual storytelling

## Design Aesthetic

Modern flat illustration with simplified shapes, solid colors, and friendly character-driven narratives. Clean vector graphics with minimal detail, focusing on clear communication through visual metaphors and relatable scenarios.

## Background

- Color: Soft Cream (#FFF9F0)
- Texture: Smooth flat color, no gradients or textures

## Typography

### Primary Font (Headlines)

Rounded sans-serif like Quicksand or Nunito. Friendly, approachable letterforms with soft curves. Medium to bold weight for titles.

### Secondary Font (Body)

Clean humanist sans-serif like Open Sans or Lato. Excellent readability with warm, friendly character.

## Visual Elements

- Simplified human characters with minimal facial features
- Flat geometric shapes with no shadows or depth
- Solid color fills, no gradients
- Simple icons representing concepts
- Decorative elements: dots, lines, simple patterns
- Illustrations should tell a story or show a scenario

## Style Rules

### Do

- Keep illustrations simple and iconic
- Use consistent line weights (or no outlines)
- Create visual hierarchy through size and color
- Show relatable human scenarios
- Use whitespace generously

### Don't

- Add realistic details or textures
- Use complex shading or lighting
- Create busy, cluttered compositions
- Use photographic elements
- Add slide numbers, footers, or logos

## Best For

Storytelling presentations, user journey maps, explainer content, marketing decks, social impact reports, educational materials
