# infographic

Data-driven visual storytelling with charts, icons, and structured information design

## Design Aesthetic

Clean information design focused on data visualization and structured content. Uses icons, charts, and visual hierarchies to make complex information digestible and engaging.

## Background

- Color: Light Gray (#F5F7FA)
- Texture: Subtle gradient or clean flat

## Typography

### Primary Font (Headlines)

Bold geometric sans-serif like Montserrat Bold or Raleway Bold. Clear hierarchy with strong weight contrast.

### Secondary Font (Body)

Clean sans-serif like Inter or Source Sans Pro. Optimized for data labels and small text.

## Visual Elements

- Clean bar charts, pie charts, line graphs
- Icon sets for visual categorization
- Progress bars and percentage indicators
- Timeline visualizations
- Comparison tables with visual emphasis
- Number callouts with large typography

## Style Rules

### Do

- Use consistent icon style throughout
- Align data visualizations to grid
- Create clear visual hierarchy
- Use color to encode meaning
- Label all data clearly

### Don't

- Overcomplicate charts
- Use 3D effects on graphs
- Mix too many chart types
- Use decorative fonts for data
- Add slide numbers, footers, or logos

## Best For

Data reports, research findings, annual reviews, market analysis, survey results, performance dashboards, comparison studies
